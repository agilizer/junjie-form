所有的请求都需要saasId和accessKey

  • htmlFormTest
  • htmlInputTest
  • statisticsTest

关于认证：
每一个请求都要提交参数saasId和accessKey。服务器判断这两个值是否相等，如果不等，直接返回状态码403.
默认的saasId和accessKey值为：
private static String saasKey="884b8165-7aad-4467-bc7e-e05e8f5efd49";
private static String accessKey="023e498c-7ff8-41cd-8874-137c46b73c5b";
